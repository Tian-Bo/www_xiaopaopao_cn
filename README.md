```js
APP-|
    |---dist: 打包文件

    |---vue.config.js: 配置文件

    |---src: 核心目录
        |---App.vue: 
        |---main.js: 
        |---api:    入口文件
        |---assets:    全局配置

        |---components:     静态资源
        |---config: 公共组件
        |---container:     路由统一管理
        |---filters:       视图文件
        |---layouts:    入口文件
        |---locale:    全局配置
        |---router:        接口统一请求管理
        |---store:     静态资源
        |---style: 公共组件
        |---views:     路由统一管理
            |---index.vue:
            |---creation:
            |---distribution:
            |---general:
            |---home:
            |---management:
            |---marketing:
            |---service:
            |---setting:
            |---shop:
            |---user:


	|---README.md:
```
